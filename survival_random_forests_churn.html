Random Forests

Random Forests (RF) is a machine learning technique which builds a large number of decision trees that: - are based on bootstrap samples. Each tree is based on a random sample with replacement of all observations. - each tree division is based on a random sample of predictors. - There is no prunning, trees are as long as possible, they are not cut

For building each RF tree a part of the observations is not used(37% aprox.). This is called out-of-bag -OOB- sample and is used for a honest estimate of the model predictive capability.

Random Survival Forest (RSF) es la extensin de RF para datos de supervivencia Propiedades de Random Forest Analiza eficientemente un gran nmero de variables, sin tener que hacer seleccin previa Es un mtodo no paramtrico, ya que no hace supuestos sobre el modelo. Puede incorporar relaciones no linealese interacciones En supervivencia, no asume el principio de riesgos proporcionales Random Forest tiene dos parmetros de tuning: el nmero de rboles y el nmero de predictores que son evaluados en cada divisin

La construccin de cada rbol de RSF se basa en el particionamiento recursivo, donde cada nodo es dividido en dos nodos hijos, seleccionando el predictor que maximiza la diferencia en supervivenciaen los nodos hijos Test log-rank El test log-rank es el mtodo estndar para evaluar si hay diferencias significativasen la funcin de supervivencia entre dos o ms grupos En cada nodo, se calcula el estadstico del test log-rank para todos los predictores evaluados en esa particin Todos los puntos de corte posibles de los predictores continuos Todas las combinaciones de categoras en los predictores categricos Se elige como mejor particin la que generan el predictor y el punto de corte con el mximo estadstico del test

Se suele establecer un nmero mnimo de eventos en los nodos terminales 40 Supervivencia. Alta Dimensionalidad Random Survival Forest. Predicciones En RSF, se estiman la funcin de supervivencia y la funcin de riesgo acumulado(cumulative hazard) para cada observacin (Kaplan-Meir) En primer lugar, se estiman en los nodos terminales de cada rbol Despus, se promedianlos valores de estas funciones en todos los rboles

Para establecer en RSF una prediccin numrica por observacin se suma la funcin de riesgo acumulada en todos los tiempos Es equivalente a un risk score, cuyos valores ms altos corresponden a las observaciones con mayor riesgo, con peor supervivencia Las predicciones para cada observacin, se pueden basar en todos los rboles o solo en los que no particip en su construccin (OOB sample) OOB ensemble cumulative hazard estimator En RSF, se define la Tasa de Error = 1 C-index

Importancia de las Variables La estimacin del error est basada en la muestra OOB Para evaluar la importancia de la variable X, se permutan aleatoriamente los valores de esa variable en la muestra OOB, y se vuelve a calcular el error La Importancia de la variable (VIMP) se define como la diferencia entre esos dos errores promediada sobre todos los rboles Si el error aumenta al permutar los valores de la variable significa que esa variable es importante, ya que se obtena un error menor con los valores observados

El coste computacional es alto Se permutan los valores de todas las variables en todos los rboles Para establecer adecuadamente la importancia de las variables en el modelo, hay que valorar el uso de un nmero grande de rboles, para que todas las variables tenga la oportunidad de participar en la construccin de bastantes rboles 42 Supervivencia. Alta Dimensionalidad Random Forest en R El paquete randomForest es el paquete habitual que se ha usado en R para construir modelos de Random Forest para problemas de clasificaciny regresin El paquete randomSurvivalForestes el que se usaba para problemas de supervivencia, pero ya no se mantiene

El paquete randomForestSRCes el nuevo paquete que integra los 2 anteriores, y por tanto permite construir modelos de clasificacin, regresin y supervivencia Permite paralelizacin El paquete ggRandomForestses un paquete que permite explorar los modelos construidos Extrae los objetos generados por randomForestSRCy genera grficos con el paquete ggplot2, pudindose usar los comandos de este paquete para modificar los grficos

La funcin rfsrc() se usa para construir un modelo de Random Forest
Parmetros formula : formula con la variable respuesta y los predictores data : data frame que contiene los datos ntree: nmero de rboles mtry: nmero de variables que entran como candidatas en cada divisin Por defecto: sqrt(p) en clasificacin y supervivencia y p/3 en regresin nodesize : nmero mnimo de observaciones en un nodo terminal (survival, 3) nsplit : nmero de puntos aleatorios a explorar en los predictores continuos importance = T : mtodo para calcular la importancia de las variables Poned importance=none si no se desea usar proximity = T : para calcular la matriz de proximidades entre observaciones

Issues with factors

  1. Families

There are four families of random forests: regression forests for continuous responses classification forests for factor responses Survival forests for right-censored survival settings competing risk survival forests for competing risk scenarios

  1. Allowable data types and issues related to factors

Data types must be real valued, integer, factor or logical however all except factors are coerced and treated as if real valued. For ordered factors, splits are similar to real valued variables.

If the factor is unordered, a split will move a subset of the levels in the parent node to the left daughter, and the complementary subset to the right daughter.
All possible complementary pairs are considered and apply to factors with an unlimited number of levels. However, there is an optimization check to ensure that the number of splits attempted is not greater than the number of cases in a node (this internal check will override the nsplit value in random splitting mode if nsplit is large enough; see below for information about nsplit)
